# ₿ Krypto-Dashboard

Live Kryptowährungskurse für GitHub Pages – kein API-Key, kein Backend.

🔗 **Live:** `https://<dein-username>.github.io/krypto-dashboard/`

## Features
- **Dashboard** – Top 100 Coins, Live-Kurse, Sparklines (7d), Marktübersicht
- **Filter** – Top 10, Gewinner, Verlierer, Watchlist, Suche
- **Watchlist** – Favoriten via localStorage
- **Konverter** – Krypto ↔ Fiat, Swap-Funktion
- **Auto-Refresh** – jede Minute
- **Kein API-Key** – CoinGecko Free Tier, ~50 Req/min

## Deployment
```
Settings → Pages → Source: main / root
```
